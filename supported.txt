Supported literals:
- Unit
- True/False
- Num (integers)
- String

Supported types:
- Concrete types:
	- bool
	- int
	- unit
	- string
- Type constructors:
	- list
	- tupple (but changed in pairs: (a,b,c,d) -> (((a,b),c),d) )

Supported patterns for match:
- simple list pattern (nil then cons or cons then nil)
- simple prod pattern 

Supported patterns for functions:
- exhaustive pattern in one clause

Not well supported:
- multi-clauses functions
- guarded patterns
